# QooERP 消息服务模块 - 应用设计文档

## 核心接口

```java
@RestController
@RequestMapping("/api/message")
public class MessageController {
    @PostMapping("/send")
    public Result<Void> send(@RequestBody MessageDTO dto);
    
    @GetMapping("/list")
    public Result<List<MessageDTO>> list();
    
    @GetMapping("/unread")
    public Result<List<MessageDTO>> unread();
}
```
