## 二、表结构

### 2.1 project（项目表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| project_no | VARCHAR | 项目编号 |
| project_name | VARCHAR | 项目名称 |
| status | TINYINT | 状态(0-草稿,1-已审批,2-进行中,3-已完成,4-已关闭) |
| start_date | DATE | 开始日期 |
| end_date | DATE | 结束日期 |
| manager_id | BIGINT | 项目经理ID |
| manager_name | VARCHAR | 项目经理姓名 |
| description | TEXT | 项目描述 |

### 2.2 project_task（任务表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| task_no | VARCHAR | 任务编号 |
| project_id | BIGINT | 项目ID |
| task_name | VARCHAR | 任务名称 |
| status | TINYINT | 状态(0-待分配,1-进行中,2-已完成) |
| assignee_id | BIGINT | 负责人ID |
| assignee_name | VARCHAR | 负责人姓名 |
| priority | TINYINT | 优先级(1-低,2-中,3-高) |
| start_date | DATE | 开始日期 |
| end_date | DATE | 结束日期 |

### 2.3 project_milestone（里程碑表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| milestone_no | VARCHAR | 里程碑编号 |
| project_id | BIGINT | 项目ID |
| milestone_name | VARCHAR | 里程碑名称 |
| status | TINYINT | 状态(0-未达成,1-已达成) |
| target_date | DATE | 目标日期 |
| actual_date | DATE | 实际日期 |
