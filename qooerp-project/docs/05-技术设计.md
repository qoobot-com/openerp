## 二、核心功能实现

### 2.1 项目进度计算

```java
@Service
public class ProjectProgressService {
    
    public ProjectProgressDTO calculateProgress(Long projectId) {
        // 计算项目进度
        long totalTasks = taskMapper.countByProjectId(projectId);
        long completedTasks = taskMapper.countCompletedByProjectId(projectId);
        
        BigDecimal progress = BigDecimal.ZERO;
        if (totalTasks > 0) {
            progress = new BigDecimal(completedTasks)
                .divide(new BigDecimal(totalTasks), 4, RoundingMode.HALF_UP)
                .multiply(new BigDecimal("100"));
        }
        
        return ProjectProgressDTO.builder()
            .totalTasks(totalTasks)
            .completedTasks(completedTasks)
            .progress(progress)
            .build();
    }
}
```
