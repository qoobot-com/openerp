# QooERP - 企业级开源ERP系统功能矩阵

QooERP将采用**四层微服务架构**，结合容器化部署和分布式系统技术，实现高可用性和可扩展性。

## 一、系统架构设计

| 层级名称 | 功能 |
|---------|------|
| 表现层 | TypeScript + Vue 3 + TDesign (PC + Mobile), Electron/Capacitor + Vite + Pinia，100+种语言，主题定制与个性化仪表盘，操作日志回放，批量操作，快捷键配置 |
| 业务逻辑层 | Spring Boot + Nacos + Ribbon + Feign + Sentinel + Nacos + Activiti + 低代码表单设计工具，API文档自动生成器，模块依赖管理器 |
| 数据访问层 | PostgreSQL主数据库，Spring Cache + Caffeine本地缓存，消息队列（数据库轮询），MinIO分布式文件存储，ShardingSphere分片管理，Druid连接池 |
| 基础设施层 | Kubernetes容器编排，HPA自动扩缩容，Istio服务网格，服务健康检查，资源调度优化 |

## 二、DDD领域分层设计

| 层次名称 | 功能模块 |
|---------|---------|
| 用户界面层 | Web前端（Vue 3 + TDesign），移动端（iOS/Android/小程序），API网关，多语言多时区，权限校验与会话管理 |
| 应用层 | 应用服务，命令处理器，查询处理器，事件订阅器，工作流编排，跨领域协调 |
| 领域层 | 聚合根，实体，值对象，领域服务，领域事件，仓储接口，领域规则引擎 |
| 基础设施层 | 仓储实现，数据访问，外部服务集成，消息队列，缓存实现，文件存储 |
