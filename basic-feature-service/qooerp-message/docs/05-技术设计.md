# QooERP 消息服务模块 - 技术设计文档

> **数据库**: PostgreSQL 15+

## 技术栈

| 技术 | 版本 |
|------|------|
| Spring Boot | 3.2.x |
| RocketMQ | 5.x |
| PostgreSQL | 15+ | 数据库 |

## 配置

```yaml
server:
  port: 8098

spring:
  application:
    name: qooerp-message
  datasource:
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:5432/qooerp_message
    username: qooerp_message
    password: qooerp_message_2026

rocketmq:
  name-server: localhost:9876
```
