# QooERP 任务调度模块 - 技术设计文档

## 技术栈

| 技术 | 版本 |
|------|------|
| Spring Boot | 3.2.x |
| XXL-JOB | 2.4.x |
| PostgreSQL | 15+ |

## 配置

```yaml
server:
  port: 8100

spring:
  application:
    name: qooerp-scheduler
  datasource:
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:5432/qooerp_scheduler
    username: qooerp_scheduler
    password: qooerp_scheduler_2026

xxl:
  job:
    admin:
      addresses: http://localhost:8080/xxl-job-admin
```
