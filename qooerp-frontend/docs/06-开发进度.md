## 一、项目概述

### 1.1 项目信息

| 项目名称 | QooERP 统一前端工程 |
|---------|-------------------|
| 项目代号 | qooerp-frontend |
| 技术栈 | Vue 3 + TypeScript + Vite + TDesign |
| 开发模式 | Monorepo (pnpm workspace) |
| 负责团队 | 前端团队 |
| 计划周期 | 2026 Q2 - 2027 Q1 |
| 当前阶段 | Phase 6: 跨端适配开发 |

---

### 1.2 项目目标

- **功能目标**: 完成 17 个业务领域,约 480+ 页面
- **性能目标**: 首屏加载 < 2s, 接口响应 < 500ms
- **质量目标**: 代码覆盖率 ≥ 80%, 线上错误率 < 1%
- **交付目标**: 支持 Web、Desktop、Mobile 三端
