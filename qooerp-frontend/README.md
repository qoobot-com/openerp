## Quick Start

### Prerequisites
- **Node.js** >= 18.17.0
- **pnpm** >= 8.0.0
- **Git** >= 2.30.0

### Install Dependencies
```bash
# Clone repository
git clone https://github.com/qoobot-com/openerp.git
cd qooerp/qooerp-frontend

# Install pnpm (if not installed)
npm install -g pnpm

# Install project dependencies
pnpm install
```

### Development
```bash
# Start Web development server
pnpm dev

# Start Desktop development (Electron)
pnpm dev:desktop

# Start Mobile development (Capacitor)
pnpm dev:mobile

# Start all platforms
pnpm dev:all
```

### Build
```bash
# Build for Web production
pnpm build

# Build Desktop app
pnpm build:desktop

# Build Mobile app
pnpm build:mobile

# Build all platforms
pnpm build:all
```

### Code Quality
```bash
# Lint check
pnpm lint

# Lint auto-fix
pnpm lint:fix

# Type check
pnpm type-check

# Unit tests
pnpm test

# E2E tests
pnpm test:e2e
```
