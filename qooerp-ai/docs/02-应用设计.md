# QooERP AI智能服务模块 - 应用设计文档

## 核心接口

```java
@RestController
@RequestMapping("/api/ai")
public class AiController {
    @PostMapping("/recommend")
    public Result<AiRecommendationDTO> recommend(@RequestBody RecommendDTO dto);
    
    @PostMapping("/predict")
    public Result<AiPredictionDTO> predict(@RequestBody PredictDTO dto);
    
    @PostMapping("/chat")
    public Result<String> chat(@RequestBody ChatDTO dto);
}
```
