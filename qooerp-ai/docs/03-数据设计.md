# QooERP AI智能服务模块 - 数据设计文档

## 数据库表

### ai_model（AI模型表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| model_name | VARCHAR | 模型名称 |
| model_type | VARCHAR | 模型类型 |
| version | VARCHAR | 版本 |
| status | TINYINT | 状态 |

### ai_recommendation（推荐记录表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID |
| recommendation_data | TEXT | 推荐数据 |
| create_time | DATETIME | 创建时间 |
