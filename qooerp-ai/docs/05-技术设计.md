# QooERP AI智能服务模块 - 技术设计文档

> **数据库**: PostgreSQL 15+

## 技术栈

| 技术 | 版本 |
|------|------|
| Spring Boot | 3.2.x |
| Spring AI | 1.0.x |
| LangChain4J | Latest |
| PostgreSQL | 15+ | 数据库 |
| pgvector | Latest | PostgreSQL 向量扩展 |

## 配置

```yaml
server:
  port: 8102

spring:
  application:
    name: qooerp-ai
  datasource:
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:5432/qooerp_ai
    username: qooerp_ai
    password: qooerp_ai_2026
  ai:
    openai:
      api-key: ${OPENAI_API_KEY}
```
