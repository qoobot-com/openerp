## 一、业务概述

### 1.1 模块定位

qooerp-user是QooERP系统的用户管理核心服务，负责：
- 用户管理（创建、编辑、删除、查询）
- 用户信息维护（基本信息、联系方式）
- 用户状态管理（启用、禁用、锁定）
- 用户与角色关联
- 用户与部门关联

### 1.2 业务目标

| 目标 | 描述 |
|------|------|
| 灵活性 | 支持灵活的用户信息配置 |
| 安全性 | 用户信息加密存储 |
| 可追溯 | 完整的用户操作日志 |
| 高可用 | 用户服务高可用 |

## 三、业务规则

### 3.1 用户规则

| 规则 | 说明 |
|------|------|
| 用户名唯一 | 用户名全局唯一 |
| 手机号唯一 | 手机号全局唯一 |
| 邮箱唯一 | 邮箱全局唯一 |
| 密码复杂度 | 至少8位，包含大小写字母和数字 |
| 账户删除校验 | 账户有数据时不可删除 |

### 3.2 用户状态规则

| 规则 | 说明 |
|------|------|
| 默认状态 | 新用户默认为启用状态 |
| 状态切换 | 禁用后用户无法登录 |
| 密码重置 | 重置密码后需首次登录修改 |
