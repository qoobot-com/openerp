> 创建日期：2026-02-17
> 服务名称：qooerp-sales
> 端口：8089

---

## 一、技术栈

| 分类 | 技术选型 | 版本 |
|------|---------|------|
| 开发语言 | Java | 17 |
| 框架 | Spring Boot | 3.2.x |
| 服务注册 | Nacos | 2.3.x |
| 持久层 | MyBatis Plus | 3.5.x |
| 数据库 | MySQL | 8.0.x |
| 缓存 | Redis | 7.0.x |

---

## 二、配置文件

```yaml
server:
  port: 8089

spring:
  application:
    name: qooerp-sales
  
  datasource:
    url: jdbc:mysql://localhost:3306/qooerp_sales
    username: root
    password: root
  
  redis:
    host: localhost
    port: 6379
    database: 3
```
