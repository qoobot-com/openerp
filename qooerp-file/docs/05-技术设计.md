# QooERP 文件服务模块 - 技术设计文档

> **数据库**: PostgreSQL 15+

## 技术栈

| 技术 | 版本 |
|------|------|
| Spring Boot | 3.2.x |
| MinIO | Latest |
| PostgreSQL | 15+ | 数据库 |

## 配置

```yaml
server:
  port: 8097

spring:
  application:
    name: qooerp-file
  datasource:
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:5432/qooerp_file
    username: qooerp_file
    password: qooerp_file_2026

minio:
  endpoint: http://localhost:9000
  access-key: minioadmin
  secret-key: minioadmin
```
