# QooERP 任务调度模块 - 数据设计文档

## 数据库表

### schedule_job（定时任务表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| job_no | VARCHAR | 任务编号 |
| job_name | VARCHAR | 任务名称 |
| cron_expression | VARCHAR | Cron表达式 |
| status | TINYINT | 状态 |

### schedule_log（任务日志表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| job_id | BIGINT | 任务ID |
| execute_time | DATETIME | 执行时间 |
| execute_result | TEXT | 执行结果 |
| status | TINYINT | 状态 |
