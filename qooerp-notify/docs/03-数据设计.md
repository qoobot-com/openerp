# QooERP 通知服务模块 - 数据设计文档

## 数据库表

### notify_record（通知记录表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| notify_no | VARCHAR | 通知编号 |
| type | VARCHAR | 通知类型 |
| receiver | VARCHAR | 接收者 |
| content | TEXT | 内容 |
| status | TINYINT | 状态 |

### notify_template（通知模板表）

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | BIGINT | 主键 |
| template_code | VARCHAR | 模板编码 |
| template_name | VARCHAR | 模板名称 |
| content | TEXT | 内容 |
| type | TINYINT | 类型 |
