# QooERP 通知服务模块 - 业务设计文档

> 模块名称: qooerp-notify  
> 模块版本: v1.0  
> 创建日期: 2026-02-17  

## 一、业务概述

通知服务模块提供统一的通知能力,支持邮件、短信、站内通知、推送通知等多种通知方式。

## 二、核心功能

- 邮件通知(邮件发送、邮件模板)
- 短信通知(短信发送、短信模板)
- 站内通知(站内消息、系统公告)
- 推送通知(APP推送、浏览器推送)

## 三、领域模型

| 实体 | 说明 |
|------|------|
| NotifyRecord | 通知记录 |
| NotifyTemplate | 通知模板 |
