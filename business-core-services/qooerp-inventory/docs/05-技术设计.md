## 一、技术栈

| 分类 | 技术选型 | 版本 |
|------|---------|------|
| 开发语言 | Java | 17 |

| 持久层 | MyBatis Plus | 3.5.x |
| 数据库 | PostgreSQL | 15+ |
| 缓存 | Redis | 7.0.x |

---

## 二、配置文件

```yaml
server:
  port: 8090

spring:
  application:
    name: qooerp-inventory

  datasource:
    driver-class-name: org.postgresql.Driver
    url: jdbc:postgresql://localhost:5432/qooerp_inventory
    username: qooerp_inventory
    password: qooerp_inventory_2026

  redis:
    host: localhost
    port: 6379
    database: 4
```
