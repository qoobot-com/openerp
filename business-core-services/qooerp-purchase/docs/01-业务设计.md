# QooERP 采购管理 - 业务设计文档

> 版本：v1.0
> 创建日期：20xx-xx-xx
> 服务名称：qooerp-purchase
> 优先级：P1-B（核心业务）
> 端口：8091

---

## 一、概述

采购管理是 QooERP 的核心业务模块之一，负责采购订单、采购入库、采购退货、采购报表等采购核心业务。

---

## 二、核心实体

```
PurchaseOrder（采购订单）
PurchaseOrderDetail（订单明细）
PurchaseReceipt（入库单）
PurchaseReceiptDetail（入库明细）
PurchaseReturn（退货单）
PurchaseReturnDetail（退货明细）
```

---

## 三、功能模块

| 模块 | 功能 |
|------|------|
| 采购订单 | 订单创建、审核、收货 |
| 采购入库 | 入库单、入库处理 |
| 采购退货 | 退货申请、退货处理 |
| 采购报表 | 采购统计、采购分析 |

---

## 四、核心表清单

| 表名 | 说明 |
|------|------|
| purchase_order | 采购订单表 |
| purchase_order_detail | 订单明细表 |
| purchase_receipt | 入库单表 |
| purchase_receipt_detail | 入库明细表 |
| purchase_return | 退货单表 |
| purchase_return_detail | 退货明细表 |
