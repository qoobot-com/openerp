## 一、模块概述

### 1.1 模块定位

qooerp-scm（供应链管理）是QooERP系统的核心业务模块之一，负责管理企业与供应商、客户之间的采购、销售、物流等业务流程。

### 1.2 核心功能

| 功能类别 | 子功能 | 说明 |
|---------|--------|------|
| 供应商管理 | 供应商档案 | 供应商基本信息管理 |
| | 供应商分类 | 供应商分类体系 |
| | 供应商评估 | 供应商质量评估 |
| | 供应商筛选 | 供应商筛选决策 |
| 客户管理 | 客户档案 | 客户基本信息管理 |
| | 客户分类 | 客户分类体系 |
| | 客户分级 | 客户等级管理 |
| 物流管理 | 物流跟踪 | 物流状态查询 |
| | 物流费用 | 运费管理 |

### 1.3 技术架构

| 层级 | 技术 | 说明 |
|------|------|------|
| 数据库 | PostgreSQL | 主数据库 |
| ORM | MyBatis-Plus | 持久层框架 |
| 缓存 | Redis | 分布式缓存 |
| 服务调用 | OpenFeign | 微服务调用 |
| 分页 | MyBatis-Plus | 分页插件 |
