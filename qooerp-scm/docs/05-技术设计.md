# QooERP 供应链管理 - 技术设计文档

> 版本：v1.0
> 创建日期：2026-02-17
> 服务名称：qooerp-scm
> 端口：8088

---

## 一、技术栈

| 分类 | 技术选型 | 版本 |
|------|---------|------|
| 开发语言 | Java | 17 |
| 框架 | Spring Boot | 3.2.x |
| 服务注册 | Nacos | 2.3.x |
| 持久层 | MyBatis Plus | 3.5.x |
| 数据库 | MySQL | 8.0.x |
| 缓存 | Redis | 7.0.x |

---

## 二、配置文件

```yaml
server:
  port: 8088

spring:
  application:
    name: qooerp-scm
  
  datasource:
    url: jdbc:mysql://localhost:3306/qooerp_scm?useUnicode=true&characterEncoding=utf8&serverTimezone=Asia/Shanghai
    username: root
    password: root
  
  redis:
    host: localhost
    port: 6379
    database: 2
```
