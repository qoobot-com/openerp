## 一、业务概述

### 1.1 模块定位

qooerp-organization是QooERP系统的组织管理核心服务，负责：
- 部门管理（创建、编辑、删除、树形结构）
- 岗位管理（岗位信息、岗位层级）
- 组织架构管理（组织树、部门关系）
- 组织权限管理（组织数据权限）

### 1.2 业务目标

| 目标 | 描述 |
|------|------|
| 灵活性 | 支持灵活的组织架构调整 |
| 可扩展性 | 支持无限层级部门 |
| 树形结构 | 清晰的组织树展示 |
| 数据权限 | 组织级别的数据权限控制 |

## 三、业务规则

### 3.1 部门规则

| 规则 | 说明 |
|------|------|
| 部门编码唯一 | 部门编码全局唯一 |
| 部门层级 | 部门最多支持10级 |
| 部门删除校验 | 部门下有用户或子部门时不可删除 |
| 默认部门 | 必须有一个默认部门 |

### 3.2 岗位规则

| 规则 | 说明 |
|------|------|
| 岗位编码唯一 | 岗位编码全局唯一 |
| 岗位级别 | 岗位级别按数字大小排序 |
